CRUISE CONTROL SYSTEM (DIAGNOSTICS) > Diagnostics with Phenomenon

DIAGNOSTIC PROCEDURE WITH PHENOMENON

Symptom

Repair area

Note

1

Cruise control main switch is not turned to ON. (Cruise indicator light does not illuminate.)

(1) Check the cruise indicator light.

(2) Check the cruise control command switch.

2

Cruise control cannot be set.

(1) Check the cruise control command switch.

(2) Check stop light & brake switch.

(3) Check clutch switch.

(4) Check the neutral switch.

(5) Check vehicle speed sensor.

3

Vehicle speed is not held within set speed ±3 km/h (±2 MPH).

Check the vehicle speed sensor.

4

Vehicle speed does not increase or does not return to set speed after RESUME/ACCEL switch has been pressed.

Check the RESUME/ACCEL switch.

5

Vehicle speed does not decrease after command switch has been pressed.

Check the command switch.

6

Cruise control is not released after CANCEL switch has been pressed.

Check the CANCEL switch.

7

Cruise control is not released after brake pedal has been depressed.

Check the stop light & brake switch.

8

Cruise control is not released after clutch pedal has been depressed.

Check the clutch switch.

9

Cruise control is not released after shifting to the neutral position.

Check the neutral switch.
