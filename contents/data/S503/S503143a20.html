MANUAL TRANSMISSION AND DIFFERENTIAL(5MT) > Oil Seal

REPLACEMENT

1. Clean the transmission exterior.

2. Using the TORX ® BIT T70, remove the drain plug, and drain the transmission gear oil completely.

3. Using the TORX ® BIT T70, tighten the transmission gear oil drain plug.

NOTE:

Use a new gasket.

Tightening torque:

44 N·m (4.5 kgf-m, 33 ft-lb) (Aluminum gasket)

70 N·m (7.1 kgf-m, 52 ft-lb) (Copper gasket)

(A)

Drain plug

(B)

Gasket

4. Remove the rear exhaust pipe and muffler.

5. Remove the propeller shaft.

6. Using the ST, remove the oil seal.

ST    398527700 PULLER ASSY

(A)

Oil seal

7. Using the ST, install the oil seal.

ST    498057300 INSTALLER

8. Install the propeller shaft.

9. Install the rear exhaust pipe and muffler.

10. Pour in gear oil and check the oil level.
