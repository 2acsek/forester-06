AUTOMATIC TRANSMISSION(4AT) > Transfer Clutch

INSTALLATION

1. Select the thrust needle bearing.

2. Install the transfer clutch assembly to the case.

(A)

Transfer clutch assembly

(B)

Extension case

3. Tighten the bolts to secure the case.

Tightening torque:

25 N·m (2.5 kgf-m, 18.1 ft-lb)

4. Install the transmission assembly to the vehicle.
