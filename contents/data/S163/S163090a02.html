1. Apply oil to the surface of the connecting rod bearings, and install the connecting rod bearings on connecting rods and connecting rod caps.

2. Install the connecting rod on crankshaft.

3. Position each connecting rod with the marking side facing forward.

4. Tighten the connecting rod cap with the connecting rod bolt or nut. Make sure the arrow on connecting rod cap faces the front during installation.

NOTE:

• Each connecting rod has corresponding cap. Make sure that they are assembled correctly by checking their matching number to prevent confusion.

• When tightening the connecting rod bolts or nuts, apply oil on the threads.

Tightening torque:

DOHC Non-turbo model

45 N·m (4.6 kgf-m, 33.2 ft-lb)

DOHC Turbo model

52 N·m (5.3 kgf-m, 38.4 ft-lb)

5. Install the oil ring expander, upper rail and lower rail by hand.

6. Install the second ring and top ring using a piston ring expander.
