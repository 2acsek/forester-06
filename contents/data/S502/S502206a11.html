AUTOMATIC TRANSMISSION(4AT) > Converter Case

INSTALLATION

1. Check the appearance of each component and clean them.

2. Press-fit the new oil seal to converter case using ST.

ST    398437700 DRIFT

3. Install the differential assembly to the case.

4. Install the right and left side retainers.

5. Install new seal pipe to converter case.

(A)

Seal pipe

6. Install new rubber pipe to converter case.

(A)

Rubber seal

7. Apply proper amount of liquid gasket to the entire matching surface of converter case.

Liquid gasket:

THREE BOND 1215 (Part No. 004403007) or equivalent

(A)

THREE BOND 1215

(B)

Rubber seal

8. Install the converter case assembly without damaging bushing and oil seal.

Tightening torque:

41 N·m (4.2 kgf-m, 30.4 ft-lb)

9. Insert the inhibitor switch connector and transmission connector to the stay.

10. Install the air breather hose.

11. Install the ATF cooler pipe.

12. Install the oil charge pipe together with an O-ring.

13. Insert the input shaft while rotating it lightly by hand, and then check the amount of protrusion.

Normal protrusion A:

50 — 55 mm (1.97 — 2.17 in)

14. Install the torque converter clutch assembly.

15. Install the transmission assembly to the vehicle.
