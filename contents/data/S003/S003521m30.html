CRUISE CONTROL SYSTEM (DIAGNOSTICS) > Diagnostic Procedure with Diagnostic Trouble Code (DTC)

DTC 35

Detected when vehicle speed feedback is impossible.

The set vehicle speed cannot be kept because of some reasons (steep uphill, parking brake is forgotten to be returned, etc.) during cruise driving.

DTC is detected when driving condition is not suitable for cruise control. Perform cruise set operation again after clearing the possible cause.
