CLUTCH SYSTEM > Operating Cylinder

ASSEMBLY

NOTE:

Apply hydraulic oil to all parts when assembling.

Recommended brake fluid

FMVSS No. 116, new DOT3 or 4 brake fluid

1. Install the piston spring to the piston.

NOTE:

The illustration below is for a Non-turbo model.

2. Insert the piston into the operating cylinder.

3. Attach the push rod to the boot.

4. Install the boot and push rod to the operating cylinder.
