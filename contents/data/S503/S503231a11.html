MANUAL TRANSMISSION AND DIFFERENTIAL(5MT) > Vehicle Speed Sensor

INSTALLATION

NOTE:

• Discard the old vehicle speed sensor after removal, use a new sensor for the installation.

• Make sure that there is no foreign matter on the sensor mounting.

• Align the tip end of the key with the key groove on the end of speedometer shaft during installation.

1. Hand tighten the vehicle speed sensor.

2. Tighten the vehicle speed sensor using suitable tools.

Tightening torque:

5.9 N·m (0.6 kgf-m, 4.3 ft-lb)

3. Connect the connector to the vehicle speed sensor.

4. Install the front, center and rear exhaust pipes, and the muffler.

5. Lower the vehicle.

6. Connect the battery ground cable to the battery.
