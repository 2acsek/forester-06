MANUAL TRANSMISSION AND DIFFERENTIAL(5MT) > Input Shaft Assembly

DISASSEMBLY

1. Remove the O-ring from the input shaft holder. Or, remove the shim of the input shaft holder.

NOTE:

• Use new O-rings.

• The number of shims used is 0 — 2.

(A)

Input shaft holder

(B)

O-ring

(C)

Input shaft holder shim

2. Wrap vinyl tape around the input shaft spline to protect the oil seal from damage.

3. Remove the inner snap ring.

ST    398663600 PLIERS

4. Fix the input shaft holder in place so that it does not move, and while hitting the end of the holder lightly with a plastic hammer, remove the input shaft.

(A)

Input shaft holder

(B)

Input shaft

5. Remove the outer snap ring. Next, remove the oil squeeze plate and straight pin.

(A)

Snap ring

(B)

Oil squeeze plate

6. Remove the snap ring.

(A)

Snap ring

7. Remove the ball bearing using the ST and a press.

NOTE:

Remove the inner snap ring before pressing.

ST    498077000 REMOVER

8. Remove the oil seal from the input shaft holder.
