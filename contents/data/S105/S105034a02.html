FUEL INJECTION (FUEL SYSTEMS)(H4SO) > Intake Manifold

ASSEMBLY

NOTE:

Use a new gasket.

When assembling the nipple, apply liquid gasket.

Liquid gasket:

THREE BOND 1105 (Part No. 004403010)

Tightening torque:

17 N·m (1.7 kgf-m, 12.5 ft-lb)

1. Install the intake manifold.

NOTE:

Use a new gasket.

Tightening torque:

8.75 N·m (0.89 kgf-m, 6.5 ft-lb)

2. Tighten the bolts which install fuel pipes on intake manifold.

Tightening torque:

6.4 N·m (0.65 kgf-m, 4.7 ft-lb)

3. Tighten the bolts which attach the pressure regulator to the intake manifold.

Tightening torque:

19 N·m (1.9 kgf-m, 14 ft-lb)

4. Connect the fuel injector pipe.

5. Connect the fuel injector pipe LH to the fuel hose, and tighten the clamp screw.

Tightening torque:

1.25 N·m (0.13 kgf-m, 0.94 ft-lb)

6. Connect the fuel injector pipe RH to the fuel hose, and tighten the clamp screw.

Tightening torque:

1.25 N·m (0.13 kgf-m, 0.94 ft-lb)

7. Install the fuel injector.

8. Tighten the bolts which install injector pipe on intake manifold.

• RH side

Tightening torque:

19 N·m (1.9 kgf-m, 14 ft-lb)

• LH side

Tightening torque:

19 N·m (1.9 kgf-m, 14 ft-lb)

9. Tighten the 2 bolts which install the fuel pipes on the intake manifold.

Tightening torque:

6.4 N·m (0.65 kgf-m, 4.7 ft-lb)

10. Install the engine harness to the intake manifold.

11. Tighten the bolts which install engine harness on intake manifold.

Tightening torque:

19 N·m (1.9 kgf-m, 14 ft-lb)

12. Hold the engine harness to the tumble generator assembly using the harness band clip.

13. Install the purge control solenoid valve.

14. Install the tumble generator valve actuator.

15. Connect the connectors (A) to fuel injector.

16. Hold the engine harness to injector pipe by harness band (B).

• RH side

• LH side

17. Install the fuel pipe protector RH.

Tightening torque:

19 N·m (1.9 kgf-m, 14 ft-lb)

18. Install the fuel pipe protector LH.

Tightening torque:

19 N·m (1.9 kgf-m, 14 ft-lb)

19. Connect the pressure regulator vacuum hose (A) to the intake manifold.

20. Install the PCV pipe.

(1) Connect the hose (A) and (B) to intake manifold.

(2) Tighten the bolts (C) through (E) in alphabetical sequence.

Tightening torque:

6.4 N·m (0.65 kgf-m, 4.7 ft-lb)

21. Install the EGR valve.

22. Install the throttle body to the intake manifold.

23. Install the ignition coil and ignitor assembly.

24. Install the engine ground terminal to intake manifold.

Tightening torque:

19 N·m (1.9 kgf-m, 14 ft-lb)
