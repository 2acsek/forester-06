BRAKE > Rear Disc Brake Assembly

INSTALLATION

1. Install the disc rotor to hub.

2. Install the support to the housing.

Tightening torque:

53 N·m (5.4 kgf-m, 39 ft-lb)

3. Apply a thin coat of Molykote M7439 to the pad clip.

4. Install the pad to support.

5. Install the caliper body to the support.

Tightening torque:

37 N·m (3.8 kgf-m, 27.3 ft-lb)

6. Connect the brake hose using the new gaskets.

Tightening torque:

18 N·m (1.8 kgf-m, 13.0 ft-lb)

CAUTION:

When connecting the brake hose, do not twist it.

7. Bleed air from the brake system.
