FUEL INJECTION (FUEL SYSTEMS)(H4DO) > Knock Sensor

INSTALLATION

1. Install the knock sensor to the cylinder block.

NOTE:

Extraction area of knock sensor cord must be positioned at a 60° angle relative to the engine rear.

Tightening torque:

24 N·m (2.4 kgf-m, 17.7 ft-lb)

(A)

Front side

2. Connect the knock sensor connector.

3. Install the secondary air pipe LH.

NOTE:

Use a new gasket.

Tightening torque:

(A) 9 N·m (0.92 kgf-m, 6.6 ft-lb)

(B) 19 N·m (1.9 kgf-m, 14 ft-lb)

4. Install the EGR pipe.

Tightening torque:

34 N·m (3.4 kgf-m, 24.6 ft-lb)

5. Install the air intake chamber.

6. Connect the battery ground cable to the battery.
