SECURITY AND LOCKS > Immobilizer Control Module

REMOVAL

NOTE:

• Prepare all registered immobilizer keys for the model with immobilizer.

• When replacing the immobilizer control module, register the immobilizer. At that time, a new immobilizer key is required. For detailed operation, refer to the “REGISTRATION MANUAL FOR IMMOBILIZER”.

• If not replacing the immobilizer control module, re-registration of a key used before is not required.

• The following removal procedure and installation location is for LHD model.

1. Disconnect the ground cable from battery.

2. Remove the instrument panel lower cover.

3. Disconnect the connector from immobilizer control module.

4. Remove the immobilizer control module.
