MANUAL TRANSMISSION AND DIFFERENTIAL(5MT) > Input Shaft Assembly

ASSEMBLY

1. Install the ball bearing to the input shaft.

NOTE:

Place the snap ring between the input shaft and ball bearing in advance. To select the appropriate snap ring, use the table in 8).

ST1    899580100 INSTALLER
ST2    399513600 INSTALLER

2. Install the snap ring to the input shaft.

(A)

Snap ring

3. Inspect the clearance between the snap ring and the ball bearing.

4. Install the straight pin and oil squeeze plate to the input shaft.

(A)

Straight pin

(B)

Oil squeeze plate

5. Install the snap ring.

6. Install the oil seal to the input shaft holder.

NOTE:

Before installing the oil seal, apply grease to the lip seal section.

ST    398507703 DUMMY COLLAR

7. Wrap vinyl tape around the shaft spline, and while hitting lightly by hand, insert the input shaft into the holder.

(A)

Vinyl tape

(B)

Input shaft

8. Install the snap ring to the input shaft holder.

NOTE:

Select an appropriate snap ring to make the clearance between the snap ring and bearing between 0 — 0.12 mm (0 — 0.0047 in)
