MANUAL TRANSMISSION AND DIFFERENTIAL(5MT) > Counter Gear

INSTALLATION

1. Install the O-ring and the straight pin to the counter gear shaft.

2. Attach the following parts to the main case (right side), and push the shaft all the way into the case.

• Counter gear shaft

• Counter gear washer × 2

• Needle bearing × 2

• Counter gear collar

• Counter gear

• Straight pin

• Snap ring

3. Install the transmission case.

4. Install the transfer case together with the extension case assembly.

5. Install the manual transmission assembly to the vehicle.

NOTE:

• Make sure that the cut off face of the counter gear shaft does not protrude from the end face of the case.
