CLUTCH SYSTEM > Clutch Pedal

INSTALLATION

1. Install the clutch pedal bracket.

Tightening torque:

18 N·m (1.8 kgf-m, 13.0 ft-lb)

2. Install the clutch master cylinder mounting nut.

Tightening torque:

18 N·m (1.8 kgf-m, 13.0 ft-lb)

3. Insert a new clevis pin to the assist rod, and install the snap pin.

NOTE:

Apply grease to the clevis pin.

4. Measure the stroke of the clutch pedal.

5. Adjust the hill holder. (Model with hill holder)

6. Connect the clutch switch connector. (Model with cruise control)

7. Install the instrument panel lower cover.
