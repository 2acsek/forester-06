COOLING(H4SO) > Thermostat

INSTALLATION

1. ENGINE SIDE

1. Install a gasket to thermostat, and install the thermostat and gasket to water pump as a unit. Then, install the thermostat cover.

NOTE:

• When reinstalling the thermostat, use a new gasket.

• The thermostat must be installed with the jiggle pin facing upward.

Tightening torque:

12 N·m (1.2 kgf-m, 8.9 ft-lb)

(A)

Thermostat cover

(B)

Gasket

(C)

Thermostat

(D)

Jiggle pin

2. Connect the radiator outlet hose to thermostat cover.

3. Install the under cover.

4. Lower the vehicle.

5. Fill engine coolant.

2. ATF WARMER SIDE

1. Attach the gasket to the thermostat, and install the thermostat and gasket to water pump together. Then, install the thermostat cover.

NOTE:

• When reinstalling the thermostat, use a new gasket.

• The thermostat must be installed with the jiggle pin facing upward.

Tightening torque:

12 N·m (1.2 kgf-m, 8.9 ft-lb)

(A)

Thermostat cover

(B)

Gasket

(C)

Thermostat

(D)

Jiggle pin

2. Connect the radiator outlet hose to the thermostat cover.

3. Install the intake manifold.

4. Fill engine coolant.
