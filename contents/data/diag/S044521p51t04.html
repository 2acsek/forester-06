CHECK OUTPUT SIGNAL FROM TCM USING SUBARU SELECT MONITOR.

1) Connect the connectors to TCM and transmission.

2) Connect the Subaru Select Monitor to the data link connector.

3) Turn the ignition switch to ON (engine OFF), and turn the Subaru Select Monitor power switch to ON.

4) Shift the select lever to the “N” range, and fully close the throttle pedal. (Vehicle speed is 0 km/h (0 MPH))

5) Read the data of transfer duty solenoid using Subaru Select Monitor.

• Transfer duty solenoid is indicated in “%”.
