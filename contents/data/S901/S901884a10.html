ENTERTAINMENT > Navigation System

INSPECTION

1. CHECK THE OPERATION OF THE NAVIGATION

Symptom

Repair order

No power coming in. (No display and no sound from speakers)

(1) Check the fuse and power supply for navigation unit.

(2) Check the ground for navigation unit.

(3) Check the ground for center monitor.

(4) Remove the navigation unit and repair it.

A specific speaker does not operate.

(1) Check the speaker.

(2) Check the output circuit between navigation unit and speaker.

Navigation generates noise while the engine is running.

(1) Check the ground for navigation unit.

(2) Check the generator.

(3) Check the ignition coil.

(4) Remove the navigation unit and repair it.

2. CHECK EACH CONNECTION

Check the connection status of each connector and make sure there is no poor contact in each terminal.

NOTE:

If there are no problems in the inspections of 1 and 2 above, use “3. Switching to Connection Inspection Mode” of the navigation system to perform the inspections indicated in “4” and later.

3. SWITCHING TO CONNECTION INSPECTION MODE

1. Turn the ignition switch to ON.

2. Press the navigation (MAP) button.

3. Press the “navigation setup” button from the TOP menu.

4. Press the “vehicle signal” button from the navigation setting screen.

5. The display will switch to the screen for checking the input signal.

4. CHECK THE GPS ANTENNA

Items to be displayed

Display contents

Repair order

GPS

• Displays the number of satellite detected (Up to 8).

• Displays the positioning status (2D/3D/ — ).

When GPS antenna is connected, the indicator illuminates.

When GPS antenna is not connected, the indicator goes off.

If the indicator is off, there could be a problem in the GPS antenna unit.

(1) Check the harness and connector between GPS antenna and navigation unit.

(2) Replace the GPS antenna.

(3) Replace the navigation unit.

5. CHECK THE BACK SENSOR

Items to be displayed

Display contents

Repair order

REV

• When selecting or shifting to the back gear, the indicator illuminates.

• When setting to other than the back gear, the indicator goes off.

(1) Check the harness and connector between back-up light switch and navigation unit. (MT model)

(2) Check the harness and connector between inhibitor switch and navigation unit. (AT model)

(3) Replace the navigation unit.

6. CHECK THE ILLUMINATION

Items to be displayed

Display contents

Repair order

ILL

• When the lighting switch is turned to ON, the indicator illuminates.

• When the lighting switch is turned to OFF, the indicator goes off.

(1) Check the harness and connector between tail & illumination relay and navigation unit.

(2) Replace the navigation unit.

7. CHECK SPEED SENSOR

Items to be displayed

Display contents

Repair order

SPD

• Displays the accumulated amount of vehicle speed pulse. (0 — 65535 ⇒ 0)

• Displays the pulse learning status.

Displays “In learning” during pulse learning and “Optimum learning” after the learning is completed.

When the vehicle speed pulse signal is normal, the indicator illuminates.

When the vehicle speed pulse signal is not normal, the indicator goes off.

When the indicator is off, a malfunction is detected where a period of time in which vehicle input pulse is not being received has continued, even though the GPS vehicle positional information continues to change. There may be an open circuit in the vehicle speed pulse signal wire.

(1) Check the harness and connector between vehicle speed sensor and navigation unit. (MT model)

(2) Check the harness and connector between TCM and navigation unit. (AT model)

(3) Replace the navigation unit.
