BRAKE > Rear Drum Brake Assembly

INSTALLATION

1. Clean the back plate and wheel cylinder assembly.

2. Install the wheel cylinder assembly on back plate, and tighten bolts.

Tightening torque:

10 N·m (1.0 kgf-m, 7.2 ft-lb)

3. Apply grease to the portions indicated by arrows in Figure.

Brake grease:

Dow Corning Molykote M 7439 (Part No. 003602001)

4. Install the brake assembly on housing, and tighten the bolts to install back plate.

Tightening torque:

53 N·m (5.4 kgf-m, 39.1 ft-lb)

5. Install the hub.

6. Connect the brake hose, and tighten the brake hose flange nut.

Tightening torque:

15 N·m (1.5 kgf-m, 11.1 ft-lb)

CAUTION:

When connecting the brake hose, do not twist it.

7. Install the ABS wheel speed sensor on back plate.

Tightening torque:

33 N·m (3.4 kgf-m, 25 ft-lb)

8. Apply grease to the adjusting screw and both ends of adjuster.

Brake grease:

Dow Corning Molykote M 7439 (Part No. 003602001)

9. Install the brake shoe.

10. Install the brake drum.

11. After installing rear drum brake assembly, bleed air from brake line.
