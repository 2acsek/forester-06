CHECK OPERATION OF RADIATOR FAN.

1) Connect the test mode connector.

2) Turn the ignition switch to ON.

3) Using the Subaru Select Monitor, check the forced operation of the radiator fan relay.

NOTE:

• When performing a forced operation radiator fan relay check using the Subaru Select Monitor, the radiator main fan and sub fan will repeat low speed revolution → high speed revolution → OFF in this order.

• Subaru Select Monitor

Refer to Compulsory Valve Operation Check Mode for detail procedures.
