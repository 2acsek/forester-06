VEHICLE DYNAMICS CONTROL (VDC) (DIAGNOSTICS) > Diagnostic Procedure with Diagnostic Trouble Code (DTC)

DTC 45 INCORRECT VDC CONTROL MODULE SPECIFICATIONS

DTC DETECTING CONDITION:

Different control module specification

TROUBLE SYMPTOM:

• ABS does not operate.

• VDC does not operate.

STEPCHECKYESNO

1.CHECK THE VDCCM SPECIFICATION.

Check the VDCCM identification mark.

Do the VDCCM identification mark and vehicle specification match?

Replace the VDCCM.

2.CHECK TCM SPECIFICATION.

Check the TCM identification mark.

Do the TCM identification mark and vehicle specification match?

Replace the TCM.

3.CHECK TCM.

1) Replace the TCM.

2) Erase the memory.

3) Perform the Inspection Mode.

4) Read the DTC.

Is the same DTC displayed again?

Defect exists in original TCM.

4.CHECK TCM.

In the current diagnosis, is the same DTC displayed again?

Go to the diagnosis corresponding to the DTC.

5.CHECK VDCCM.

1) Install the original TCM.

2) Replace the VDCCM.

3) Erase the memory.

4) Perform the Inspection Mode.

5) Read the DTC.

Is the same DTC displayed again?

Defect exists in original VDCCM.

6.CHECK VDCCM.

Is the same DTC as current diagnosis still displayed?

Replace the TCM.

Proceed with the diagnosis corresponding to the DTC.
