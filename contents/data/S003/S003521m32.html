CRUISE CONTROL SYSTEM (DIAGNOSTICS) > Diagnostic Procedure with Diagnostic Trouble Code (DTC)

DTC 34

The vehicle has been driven at a speed higher than the set speed for a long time (approximately 10 minutes) during cruise driving.

DTC is detected when the vehicle has been driven for a long time at a speed higher than the cruise set available vehicle speed by operating the accelerator pedal. It is normal if DTC is not detected after performing the cruise control setting again.
