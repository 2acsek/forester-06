FUEL INJECTION (FUEL SYSTEMS)(H4DOTC) > Knock Sensor

INSTALLATION

1. Install the knock sensor to the cylinder block.

Tightening torque:

24 N·m (2.4 kgf-m, 17.4 ft-lb)

NOTE:

The portion of the knock sensor cord that is pulled out must be positioned at a 60° angle relative to the engine rear.

(A)

Front side

2. Connect the knock sensor connector.

3. Install the intercooler.

4. Install the collector cover.

5. Connect the battery ground cable to the battery.
