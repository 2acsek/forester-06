IMMOBILIZER (DIAGNOSTICS) > Basic Diagnostic Procedure

PROCEDURE

STEPCHECKYESNO

1.CHECK IMMOBILIZER INDICATOR LIGHT.

1) Turn the ignition switch to OFF or ACC position.

2) Wait at least 60 seconds.

Does the immobilizer indicator light blink?

Check the immobilizer indicator light circuit.

2.CHECK KEY SWITCH.

Remove the key from ignition switch.

Does the immobilizer indicator light begin to blink within 1 second after the key is removed?

Check the key switch circuit.

3.CHECK IMMOBILIZER INDICATOR LIGHT.

Turn the ignition switch to ON.

Does the immobilizer indicator light go off?

Perform teaching operation on all keys of the vehicle.

4.CHECK ENGINE START.

Turn the ignition switch to the START position.

Does the starter motor operate?

Check the starter motor circuit.

5.CHECK ENGINE START.

Turn the ignition switch to the START position.

Does the engine start?

Immobilizer system is normal.

6.CHECK INDICATION OF DTC ON SCREEN.

1) Turn the ignition switch to OFF.

2) Connect the Subaru Select Monitor to the data link connector.

3) Turn the ignition switch and Subaru Select Monitor power switch to ON.

4) Read DTCs on the display.

Is the DTC displayed on screen?

Perform diagnostics for engine starting failure.

7.PERFORM DIAGNOSIS.

1) Inspect using the “Diagnostic Procedure with Diagnostic Trouble Code (DTC)”.

2) Repair the trouble cause.

3) Perform the Clear Memory Mode.

4) Read DTC’s again.

Is the DTC displayed on screen?

Inspect using the “Diagnostic Procedure with Diagnostic Trouble Code (DTC)”.

Finish the diagnosis.
