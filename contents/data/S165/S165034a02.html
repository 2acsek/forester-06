FUEL INJECTION (FUEL SYSTEMS)(H4DO) > Intake Manifold

ASSEMBLY

NOTE:

When assembling the nipple, apply liquid gasket.

Liquid gasket:

THREE BOND 1105 (Part No. 004403010)

Tightening torque:

17 N·m (1.7 kgf-m, 12.5 ft-lb)

1. Tighten the bolts which install fuel pipes on intake manifold.

Tightening torque:

6.4 N·m (0.65 kgf-m, 4.7 ft-lb)

2. Connect the fuel injector pipe LH to the fuel hose, and tighten the clamp screw.

Tightening torque:

1.25 N·m (0.13 kgf-m, 0.94 ft-lb)

3. Connect the fuel injector pipe RH to the fuel hose, and tighten the clamp screw.

Tightening torque:

1.25 N·m (0.13 kgf-m, 0.94 ft-lb)

4. Install the fuel injector.

5. Tighten the bolts which install injector pipe on intake manifold.

• RH side

Tightening torque:

19 N·m (1.9 kgf-m, 13.7 ft-lb)

• LH side

Tightening torque:

(A) 6.4 N·m (0.65 kgf-m, 4.7 ft-lb)

(B) 19 N·m (1.9 kgf-m, 13.7 ft-lb)

6. Install the engine harness onto intake manifold.

7. Tighten the bolts which install engine harness on intake manifold.

Tightening torque:

6.4 N·m (0.65 kgf-m, 4.7 ft-lb)

8. Connect fuel injector connector (B).

9. Secure the engine harness to the injector pipe with harness band (A).

10. Install the fuel pipe protector RH.

Tightening torque:

19 N·m (1.9 kgf-m, 13.7 ft-lb)

11. Install the fuel pipe protector LH.

Tightening torque:

19 N·m (1.9 kgf-m, 13.7 ft-lb)

12. Install the secondary air combination valve.

13. Install the purge control solenoid valve.

14. Install the EGR valve.

15. Install the throttle body to the intake manifold.

16. Install the PCV pipe.

(1) Connect the hoses (A) and (B) to intake manifold.

(2) Tighten the bolts (C) through (E) in alphabetical sequence.

Tightening torque:

6.4 N·m (0.65 kgf-m, 4.7 ft-lb)

17. Install the secondary air duct.

Tightening torque:

6.4 N·m (0.65 kgf-m, 4.7 ft-lb)

18. Install the engine ground terminal to the intake manifold.

Tightening torque:

19 N·m (1.9 kgf-m, 13.7 ft-lb)
