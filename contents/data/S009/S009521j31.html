IMMOBILIZER (DIAGNOSTICS) > Diagnostic Procedure with Diagnostic Trouble Code (DTC)

DTC P0513 INCORRECT IMMOBILIZER KEY

DTC DETECTING CONDITION:

Incorrect immobilizer key (Use of unregistered key in IMM ECM)

STEPCHECKYESNO

1.PERFORM REGISTRATION ON IGNITION KEY.

Perform registration on all keys of the vehicle. Refer to REGISTRATION MANUAL FOR IMMOBILIZER (Pub. No. S0820GZ).

Is registration for all keys complete?

END.

Replace all ignition keys (including the transponder).

2.PERFORM REGISTRATION ON IGNITION KEY.

Perform registration on all keys of the vehicle. Refer to REGISTRATION MANUAL FOR IMMOBILIZER (Pub. No. S0820GZ).

Is registration for all keys complete?

END.

Replace the IMM ECM and then replace all ignition keys (including the transponder). Execute the registration procedure next. Refer to REGISTRATION MANUAL FOR IMMOBILIZER (Pub. No. S0820GZ).
