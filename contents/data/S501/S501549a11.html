CONTROL SYSTEMS > Select Cable

INSTALLATION

1. Install the select cable to plate assembly.

Tightening torque:

18 N·m (1.8 kgf-m, 13.0 ft-lb)

2. Install the select cable to the range select lever.

3. Install the plate assembly to the transmission.

Tightening torque:

T: 24.5 N·m (2.5 kgf-m, 18.1 ft-lb)

6. Install the heat shield cover. (If equipped)
