AIRBAG SYSTEM (DIAGNOSTICS) > Diagnostic Chart with Trouble Code

DTC 22

WIRING DIAGRAM:

• LHD model

• RHD model

This code is indicated when the front airbag, pretensioner and lap pretensioner activate.

Once this code is displayed, the memory cannot be cleared. Therefore replace the following parts.

• Airbag control module

• Driver’s airbag module

• Passenger’s airbag module

• Front sub sensor of both sides

• Front seat belt outer with pretensioner of both sides.
