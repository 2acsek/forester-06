CHECK INPUT SIGNAL FOR TCM USING SUBARU SELECT MONITOR.

1) Connect the connectors to TCM and ECM.

2) Connect the Subaru Select Monitor to the data link connector.

3) Start the engine, and turn the Subaru Select Monitor power switch to ON.

4) Run the engine at idle.

5) Read the data of engine speed using Subaru Select Monitor.

Display shows engine speed signal value sent from ECM.
