FUEL INJECTION (FUEL SYSTEMS)(H4DOTC) > Intake Manifold

INSTALLATION

1. Install the intake manifold onto cylinder heads.

NOTE:

Use a new gasket.

Tightening torque:

25 N·m (2.5 kgf-m, 18.1 ft-lb)

2. Connect the fuel delivery hose, return hose, and evaporation hose.

(A)

Fuel delivery hose

(B)

Return hose

(C)

Evaporation hose

3. Connect the connector to the engine coolant temperature sensor (A), oil pressure switch (B) and crankshaft position sensor (C).

4. Connect the connector to the knock sensor.

5. Connect the connectors to camshaft position sensor.

6. Connect the connector to the ignition coil.

7. Connect the engine harness with clip (A) to the bracket.

• RH side

• LH side

8. Connect the engine harness connector.

9. Connect the brake booster vacuum hose.

10. Connect the engine coolant hoses to throttle body.

11. Connect the emission hose to the PCV valve.

12. Connect the pressure hose to the intake duct.

13. Install the power steering pump.

(1) Install the reservoir tank to the bracket.

(2) Connect the power steering pump switch to the connector.

(3) Install the power steering pump, and tighten the bolts.

Tightening torque:

22 N·m (2.2 kgf-m, 15.9 ft-lb)

(4) Install the power steering pipe bracket onto the right side intake manifold.

(5) Install the front side V-belt.

14. Install the coolant filler tank.

15. Install the intercooler.

16. Install the air cleaner element.

17. Install the air cleaner upper cover and air intake duct as a unit.

18. Connect the connector to the fuel pump relay.

19. Install the passenger’s side front side sill cover.

20. Install the glove box.

21. Install the collector cover.

22. Connect the battery ground cable to the battery.

23. Lift up the vehicle.

24. Install the under cover.

25. Fill engine coolant.
