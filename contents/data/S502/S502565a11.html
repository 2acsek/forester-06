AUTOMATIC TRANSMISSION(4AT) > ATF Cooler Pipe and Hose

INSTALLATION

1. ALL MODELS EXCLUDING THOSE WITH ATF COOLERS (WITH WARMER FEATURE)

1. Install the ATF cooler inlet and outlet pipes with the new washers.

Tightening torque:

T1: 25 N·m (2.5 kgf-m, 18.1 ft-lb)

T2: 40 N·m (4.1 kgf-m, 30 ft-lb)

T3: 45 N·m (4.6 kgf-m, 33 ft-lb)

2. Install the ATF cooler pipe to frame.

3. Connect the ATF cooler hose to the pipe on the transmission side.

NOTE:

• Install so that the hose is not folded over, excessively bent or twisted.

• Insert the hose to the specified position.

4. Connect the ATF cooler hose to the pipe on the radiator side.

5. Install the under cover.

6. Install the battery and washer tank.

7. Fill ATF.

NOTE:

Make sure there are no ATF leaks in joints between the transmission, radiator, pipes, and hoses.

2. ALL MODELS WITH ATF COOLERS (WITH WARMER FEATURE)

1. Install the ATF cooler inlet and outlet pipes with the new washers.

Tightening torque:

T1: 40 N·m (4.1 kgf-m, 30 ft-lb)

T2: 41 N·m (4.2 kgf-m, 30.4 ft-lb)

T3: 45 N·m (4.6 kgf-m, 33 ft-lb)

T4: 21 N·m (2.1 kgf-m, 15.5 ft-lb)

(A)

ATF cooler outlet pipe

(B)

ATF cooler inlet pipe

2. Install the ATF cooler assembly to the transmission.

NOTE:

Insert the tip of the ATF cooler bracket to the hole of the converter case.

Tightening torque:

T1: 21 N·m (2.1 kgf-m, 15.5 ft-lb)

T2: 23 N·m (2.3 kgf-m, 17.0 ft-lb)

T3: 33 N·m (3.4 kgf-m, 25 ft-lb)

3. Install the oil charge pipe.

4. Connect the inlet and outlet hose.

(A)

ATF inlet hose

(B)

ATF outlet hose

5. Install the inhibitor switch and transmission harness connector to the stay, and then connect the harness connector.

6. Connect the air breather hose.

7. Install the air intake chamber.
