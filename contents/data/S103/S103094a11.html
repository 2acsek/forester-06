MECHANICAL(H4SO) > Valve Rocker Assembly

INSTALLATION

1. Valve rocker assembly installation

(1) Temporarily tighten the bolts in the sequence of (a) — (d) according to the diagram.

CAUTION:

Do not damage the knock pin with the valve rocker assembly.

(2) Tighten the bolts (e) through (h) to specified torque.

(3) Tighten the bolts (a) through (d) to specified torque.

Tightening torque:

25 N·m (2.5 kgf-m, 18.4 ft-lb)

2. Install the cam sprocket.

3. Install the timing belt.

4. Adjust the valve clearance.

5. Install the rocker cover and rocker cover gasket, and then connect PCV hose.

NOTE:

Use a new rocker cover gasket and rocker cover washers.

6. Install the timing belt cover.

7. Install the crank pulley.

8. Install the V-belts.
