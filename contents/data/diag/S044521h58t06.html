CHECK INPUT SIGNAL FOR TCM USING SUBARU SELECT MONITOR.

1) Connect all connectors.

2) Connect the Subaru Select Monitor to the data link connector.

3) Lift up the vehicle and place it on rigid racks.

NOTE:

Raise all wheels off the floor.

4) Turn the ignition switch to ON, and the Subaru Select Monitor power switch to ON.

5) Start the engine.

6) Read the data of vehicle speed using Subaru Select Monitor.

• Compare the speedometer with Subaru Select Monitor indications.

• Vehicle speed is indicated in “km/h” or “MPH”

7) Slowly increase the vehicle speed to 60 km/h or 37 MPH.

NOTE:

The speed difference between front and rear wheels may illuminate the ABS warning light, but this does not indicate a malfunction. When AT control diagnosis is finished, perform the ABS memory clearance procedure of on-board diagnostics system.
