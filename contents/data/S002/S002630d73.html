AIRBAG SYSTEM (DIAGNOSTICS) > Diagnostic Chart with Trouble Code

DTC 55

This code is indicated when the side airbag and pretensioner activates.

When this code is displayed, the memory cannot be cleared. Replace the following parts.

• Airbag control module

• Front seat with side airbag module (Operating side)

• Side airbag sensor (Operating side)

• Front outer seat belt with pretensioner (Operating side)
