CHECK OUTPUT SIGNAL FROM TCM USING SUBARU SELECT MONITOR.

1) Shift the select lever to “D” range.

2) Read the data of transfer duty solenoid using Subaru Select Monitor.

• Transfer duty solenoid is indicated in “%”.

QUESTION

Is the measured value approximately 18 to 35%?
