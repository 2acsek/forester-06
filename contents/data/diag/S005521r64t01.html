CHECK OUTPUT OF ABS WHEEL SPEED SENSOR USING SUBARU SELECT MONITOR.

1) Select {Current Data Display & Save} in Subaru Select Monitor.

2) Read the ABS wheel speed sensor output corresponding to the faulty area in the Subaru Select Monitor data display mode.
