CONTROL SYSTEMS > Select Cable

ADJUSTMENT

1. Set the vehicle on a lift.

2. Set the select lever to “N” range.

3. Lift the vehicle.

4. Remove the rear exhaust pipe and muffler.

5. Remove the heat shield cover. (If equipped)

6. Loosen the adjusting nuts on both sides.

(A)

Adjusting nut A

(B)

Adjusting nut B

7. Turn the adjusting nut B until it lightly touches the connector.

(A)

Front side

(B)

Select lever

(C)

Connector

(D)

Adjusting nut B

(E)

Contact point

(F)

Adjusting nut A

8. Set a spanner wrench to adjusting nut B so that it does not rotate, and then tighten the adjusting nut A.

Tightening torque:

7.5 N·m (0.76 kgf-m, 5.5 ft-lb)

(A)

Adjusting nut A

(B)

Adjusting nut B

9. After completion of fitting, make sure that the select lever operates smoothly all across the operating range.

10. Install in the reverse order of removal.
