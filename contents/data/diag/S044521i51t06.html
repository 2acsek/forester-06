CHECK INPUT SIGNAL FOR TCM USING SUBARU SELECT MONITOR.

1) Connect the connectors to TCM and transmission.

2) Connect the Subaru Select Monitor to the data link connector.

3) Turn the ignition switch to ON, and the Subaru Select Monitor power switch to ON.

4) Start the engine.

5) Move the select lever to “P” or “N” range.

6) Read the data of turbine speed using Subaru Select Monitor.

Compare the tachometer with Subaru Select Monitor indications.
