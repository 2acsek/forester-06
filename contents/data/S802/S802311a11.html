AIRBAG SYSTEM > Side Airbag Module

INSTALLATION

CAUTION:

Refer to “CAUTION” of “General Description” before handling the airbag module.

1. Make sure that there is no foreign matter on side airbag module.

2. Install the side airbag module (A) to backrest frame assembly.

Tightening torque:

5.9 N·m (0.59 kgf-m, 4.4 ft-lb)

3. Install the side airbag harness to backrest frame assembly.

NOTE:

Secure the harness to its original position of seat.

4. Install the backrest cover.

5. Install the side airbag harness to the slide rail IN (A), and secure the connector to the bracket on the back surface of seat cushion assembly.

6. Recline the seat or slide it backward and forward, and check there is no contact between the seat backrest assembly, cushion cover assembly and side airbag harness.
